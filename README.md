# BodyAtlas
an interactive learning app for human body where you can see the human body on your screen and interact with it Most people know:
"The heart pumps blood."
"The lungs help you breathe."
"Insulin helps diabetes."

But very few people understand:

How blood actually moves through the body.
Why a heart attack happens.
What asthma does inside the lungs.
How medicines travel and work.

An interactive visual explanation can make those concepts click instantly.
you can explore the Nervous system
Digestive system
Respiratory system
Circulatory system
Skeletal system

What changes inside the body
Symptoms
Risk factors
Prevention
